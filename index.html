Тетрис для моей дипломной работы

Я разработал игровое приложение Тетрис, используя язык программирования Python и библиотеку Pygame. Основной целью было создать классическую версию известной игры, для моей дипломной работы

- Обработка событий:
Используя Pygame, я реализовал обработку событий для контроля движения тетромино. Игроки могут перемещать фигуры влево, вправо и вращать их по нажатию соответствующих клавиш.

- Графика и анимация:
Я создал простой, но привлекательный интерфейс с использованием Pygame для рисования графических элементов. Каждое тетромино отображается в разных цветах, что делает игру более визуально привлекательной. Также добавлены простые анимации при исчезновении линий.

- Система счета:
В приложении реализована система подсчета очков. Игроки зарабатывают очки за каждую удаленную линию, и счет обновляется в реальном времени на экране.

- Оптимизация и производительность:
Я использовал оптимизированные алгоритмы для обработки коллизий и удаления линий, что обеспечивает плавность игры.
Регулярные обновления экрана и управление частотой кадров позволили достичь стабильной производительности на различных устройствах.

Сайт 3д печати

На сайте, посвященном 3D печати, реализована современная архитектура, которая обеспечивает удобство и функциональность для пользователей, желающих создать и заказать 3D модели.

Технологический стек:

- Front-end: Реализован с использованием HTML, CSS и JavaScript. Для динамического контента и интуитивного пользовательского интерфейса применяются фреймворки такие как React или Vue.js. Используются адаптивные стили для обеспечения корректного отображения на различных устройствах.

- Back-end: Серверная часть написана на Node.js с использованием Express для обработки запросов. Реализована RESTful API, что позволяет эффективно взаимодействовать между клиентом и сервером.

- База данных: Для хранения данных о пользователях, проектах и выполненных заказах используется MongoDB, что обеспечивает гибкость и возможность масштабирования.

Функциональность:

- Загрузка моделей: Пользователи могут загружать свои 3D модели в формате STL или OBJ. Реализована проверка на корректность файлов для предотвращения ошибок во время печати.

- Калькулятор стоимости: Пользователи могут рассчитать стоимость печати моделей в зависимости от используемых материалов и размеров. Этот функционал реализован через серверные расчеты и динамическое обновление интерфейса.

- Интерактивная галерея: Предоставляется возможность просматривать галерею работ с возможностью фильтрации по категориям и стилям. Используется AJAX для загрузки данных без перезагрузки страницы.

- Обратная связь: Для улучшения взаимодействия с пользователями интегрирована система комментариев и оценок, что позволяет получать обратную связь и улучшать сервис.

Безопасность и производительность:

- Безопасность: Все данные передаются по защищённому протоколу HTTPS. Реализованы механизмы аутентификации и авторизации для защиты пользовательских данных.

- Оптимизация: Сайт оптимизирован для быстрого загрузки. Используются методы кэширования и минимизации ресурсов.
